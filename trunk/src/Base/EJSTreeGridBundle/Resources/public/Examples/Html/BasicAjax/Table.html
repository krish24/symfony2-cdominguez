Positions, static and dynamic spanning, types and formats

Demonstration of various layout positions, column and row spanning, cell types and formats.
On top there is one row for both grouping and searching. Individual filter row is at bottom. Toolbar is moved between head and body.
There are three headers, two on top, one at bottom. Some header show spanned captions. It is still possible to move and resize columns, sort by columns.
The rows 4 and 18 show (vertical) row spanning, also with combination of column spanning. The rows 8 and 12 show html in whole section instead of cells, the row Liberec shows merging spanned cell.
There various setting of cell types and formats, for whole column or for individual cell. Also show right button in cell.

You can dynamically span and split selected cells.
Select cells by mouse left button and click to Span icon on toolbar to span the cells or to Split button to split spanned cells, if any.
Select the cells by right mouse button to span them automatically. Click to spanned cell by middle mouse button to split cells.
